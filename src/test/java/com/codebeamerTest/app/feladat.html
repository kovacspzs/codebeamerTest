A feladat a következő lenne: Töltsd le kérlek, és telepítsd a codeBeamer 9.5.0 próbaverzióját a következő url-ről https://intland.com/download-codebeamer/. Telepítéskor válaszd a 1. Create a new codeBeamer ALM instance 2a. Create new account 2b. Skip (7 nap próbaidőd lesz) 3. Built-in 4. Ami tetszik 5. Disable local SVN services 6. Skip Írj automata tesztet Java nyelven, Selenide (https://selenide.org/) keretrendszer felhasználásával egy szabadon választott böngészővel. A böngésző incognito módban induljon. A teszt lépései a következők: 1. Lépj be a telepítéskor létrehozott felhasználóval 2. Kattints a Projects menure 3. Kattints a New Project gombra (+ icon) (A videóban Create project van) 4. Válaszd a Create a new project opciót 5. Töltsd fel az Intland_Software's_Scrum_Template.zip fájlt 6. Kattints a Next gombra 7. Jegyezd meg a Name mező értékét 8. Kattints a Finish gombra 10. Ellenőrizd az adott nevű projekt sikeresen létrejött-e (test_steps.mp4 videón a fentebb leírt lépések) Ha marad időd automatizáld kérlek ezt a pár lépést is: 1. Projekt kreálás után az úgynevezett Project Wiki lapon landolunk. Itt kattints a New Child Page ikonra 2. A felugró ablakon töltsd ki a Name mezőt tetszőleges értékkel 3. Mentsd el a Wiki lapot a Save gombra kattintással 4. Ellenőrizd, hogy létrejött-e a megadott nevű Wiki lap (bonus.mp4 videón a fentebb leírt lépések) Kérlek úgy kezeld a feladatot, mintha egy valódi greenfield project lenne. Sikeres megoldás esetén egy újabb kört fogunk szervezni ahol megnézzük a feladat magoldását, hogy lássuk milyen gondolatmenet vezetett végig a feladaton. A kódot értelmezni kell tudni, és válaszolni az esetlegesen felmerülő kérdésekre. 2020. feb. 11. éjfélig kérlek, küldd el a megoldásod zipelt git bundle vagy workspace formában. Ha esetleg bármilyen problémád van a feladattal kapcsolatban, szívesen válaszolok a kérdéseidre.
